**Before start flashing.....**

- Backup your data (important)
- Use latest OrangeFox.zip as recovery.
- For OFOX zip, [here](https://t.me/murtazasapollocommunity).
- Clean flash is required if coming from another rom.
- Always dirty flash at your own risk. I won't be responsible for any data loss.
----

## Gapps variant

**Clean flash:**
- Download the ROM
- Boot into Ofox recovery
- Wipe Dalvik/Art Cache, cache and data
- Flash the ROM
- Tap on format data, type yes and reboot to system

**Dirty flash:**
- Download the ROM
- Boot into Ofox recovery
- Wipe Dalvik/ArtCache and Cache (don't wipe data)
- Flash the ROM
- Reboot To System

----

## Vanilla variant

**Clean flash:**
- Download the ROM
- Boot into Ofox recovery
- Wipe Dalvik/Art Cache, cache and data
- Flash the ROM
- Flash gapps , microg ...etc (optional)
- Tap on format data, type yes and reboot to system

**Dirty flash:**
- Download the ROM
- Boot into  Ofox recovery
- Wipe Dalvik/ArtCache and Cache
- Flash the ROM
- Flash gapps , microg ...etc (optional)
- Reboot To System
